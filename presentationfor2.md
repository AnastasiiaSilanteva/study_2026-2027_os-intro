## **Основные понятия Git**

| Термин | Описание |
|--------|----------|
| Хранилище | Место хранения версий файлов |
| Commit | Фиксация изменений с комментом |
| История | Последовательность коммитов |
| Рабочая копия | Локальная версия файлов |
| Ветки | Изолированная разработка функций |
---

## **Ключевые команды Git**

- `git init` – создание репозитория
- `git clone` – копирование удалённого репозитория
- `git add` / `git commit` – добавление и фиксация
- `git push` / `git pull` – синхронизация с удалённым репозиторием
- `git branch` / `git merge` – управление ветками
- `git log` / `git status` – просмотр состояния
