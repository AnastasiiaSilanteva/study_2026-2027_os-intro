## 4. Инициализация git-flow

### 4.1 Запуск git-flow init

В корне репозитория выполняю:


git flow init


Появляются ветки master и develop, активна develop.

### 4.2 Отправка всех веток на GitHub


git push --all

Теперь на GitHub появятся ветки master и develop.
